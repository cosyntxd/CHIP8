# CHIP8
Yet another CHIP-8 interpreter

# Build and Run:
```properties
git clone https://github.com/cosyntxd/CHIP8
cd CHIP8
cargo run --release
```
# Roms
See a list of ROMs for CHIP8 [here](https://github.com/loktar00/chip8/tree/master/roms), download one, then drag and drop the file into the window
